# (C# 코딩) SimplePaint

## 개요
- C# 프로그래밍 학습
- 1줄 소개: C#으로 구현한 그림판 
- 사용한 플랫폼:
  - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - label, button,trackbar, picturebox, Combobox
- 사용한 기술과 구현한 기능
  - Combobox에서 색상 선택 기능 구현
  - Trackbar에서 선굵기 선택 기능 구현
  - Button에서 도형 선택 기능 구현
